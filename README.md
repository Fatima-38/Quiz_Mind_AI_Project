# 🧠 QuizMind.AI_Project

QuizMind AI is a full-stack web application that generates AI-powered multiple choice questions on Computer Science topics including DSA, OOP, DBMS, Operating Systems, and more. Built with FastAPI and React, it features instant feedback, score tracking, and bookmark functionality to help students prepare for exams effectively.

The application leverages the Groq API with LLaMA 3.3 70B model to dynamically generate high-quality questions at varying difficulty levels — Easy, Medium, and Hard. With a clean and modern dark-themed UI, QuizMind AI provides an engaging and seamless learning experience for CS students and professionals alike.
